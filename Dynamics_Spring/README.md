------

操作方法：

按顺序选中骨骼链,点击 1 按钮，然后 2 按钮添加弹簧控制器。
------

注意：

确保被选中骨骼链至少有一个上级骨骼不添加弹簧。如：一个 7 节骨骼链，不要选中第一节，应该从第二节骨骼开始。
